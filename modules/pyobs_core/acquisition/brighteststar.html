6.1.1.1. BrightestStarAcquisition¶

class pyobs.modules.acquisition.BrightestStarAcquisition(telescope: Union[str, pyobs.interfaces.ITelescope.ITelescope], camera: Union[str, pyobs.interfaces.ICamera.ICamera], target_pixel: Tuple = None, attempts: int = 5, tolerance: float = 1, *args, **kwargs)¶

Module for acquiring telescope on brightest star in field.

__init__(telescope: Union[str, pyobs.interfaces.ITelescope.ITelescope], camera: Union[str, pyobs.interfaces.ICamera.ICamera], target_pixel: Tuple = None, attempts: int = 5, tolerance: float = 1, *args, **kwargs)¶

Acquire on brightest star in field..

Parameters

  • telescope – Name of ITelescope.

  • camera – Name of ICamera.

  • target_pixel – (x, y) tuple of pixel that the star should be positioned on. If None, center of image is used.

  • attempts – Number of attempts before giving up.

  • tolerance – Tolerance in position to reach.

acquire_target(ra: float, dec: float, exposure_time: int, *args, **kwargs)¶

Acquire target at given coordinates.

Parameters

  • ra – Right ascension of field to acquire.

  • dec – Declination of field to acquire.

  • exposure_time – Exposure time for acquisition.
