3rd party packages¶

pyobs requires Python 3.7 or later and depends on a couple of amazing external packages (in alphabetical order):

  • Astroplan

  • Astropy

  • Astroquery

  • Cython

  • LMFIT

  • matplotlib

  • NumPy

  • PyYAML

  • Pandas

  • Paramiko

  • photutils

  • py_expression_eval

  • PyQt5

  • python_telegram_bot

  • Requests

  • SciPy

  • SEP

  • SleekXMPP

  • Tornado

Thanks to everyone putting time and efforts into these (and other!) open source projects!
